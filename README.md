# Strain Gauge Calculator — PWA

این پکیج شامل نسخه PWA (Progressive Web App) از محاسبه‌گر کرنش‌سنج است که قابلیت نصب روی صفحه اصلی آیفون را دارد.

## محتوای پکیج
- `index.html` — برنامه اصلی
- `manifest.json` — تنظیمات PWA (نام، آیکون، رنگ)
- `sw.js` — Service Worker برای کارکرد آفلاین
- `icons/` — آیکون‌ها در سایزهای مختلف

## مراحل نصب روی آیفون (ضروری: نیاز به هاست دارد)

⚠️ نکته مهم: سافاری در iOS اجازه نمی‌دهد فایل‌های باز شده مستقیم از حافظه گوشی (file://) با "Add to Home Screen" به‌صورت PWA کامل (آیکون + تمام‌صفحه + آفلاین) نصب شوند. باید فایل‌ها را روی یک آدرس اینترنتی (حتی رایگان) میزبانی کنید. ساده‌ترین راه‌ها:

### روش ۱ — GitHub Pages (رایگان)
1. یک ریپازیتوری جدید در GitHub بسازید.
2. تمام فایل‌های این پکیج (با حفظ پوشه `icons`) را در آن آپلود کنید.
3. در تنظیمات ریپازیتوری → Pages → فعال کنید (شاخه main، روت /).
4. آدرسی مثل `https://username.github.io/repo-name/` به شما داده می‌شود.

### روش ۲ — هر هاست استاتیک دیگر
Netlify, Vercel, Cloudflare Pages یا حتی هاست ساده‌ی شخصی هم کار می‌کند؛ فقط کافیست همه فایل‌ها با همان ساختار پوشه آپلود شوند.

## نصب نهایی روی آیفون
1. لینک هاست‌شده را در **Safari** باز کنید (مرورگرهای دیگر مثل کروم این قابلیت را در iOS ندارند).
2. روی دکمه Share (مربع با فلش رو به بالا) بزنید.
3. گزینه **Add to Home Screen** را انتخاب کنید.
4. نام را تأیید کنید و Add بزنید.

اپ با آیکون اختصاصی (همان آیکون رزت سه‌گیج کرنش‌سنج) روی صفحه اصلی ظاهر می‌شود و به‌صورت تمام‌صفحه (بدون نوار آدرس سافاری) باز می‌شود.
