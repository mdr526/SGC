# Strain Gauge Calculator — نسخه PWA

این پوشه نسخه‌ی نصب‌شدنی (PWA) محاسبه‌گر کرنش‌سنج است — برای اندروید و آیفون.

## محتویات
- `index.html` — فایل اصلی برنامه
- `manifest.json` — مشخصات نصب برنامه (نام، آیکون، رنگ تم)
- `service-worker.js` — فعال‌کننده‌ی کارکرد آفلاین و کش کردن برنامه
- `icons/` — آیکون‌های برنامه در همه‌ی سایزهای لازم (ساخته‌شده از آیکون اختصاصی، بدون پس‌زمینه سیاه، گوشه‌ها تخت برای تطبیق با ماسک گرد iOS/Android)

## نکته‌ی مهم — قابلیت نصب فقط روی HTTPS کار می‌کند
مرورگرها Service Worker (و در نتیجه نصب PWA) را فقط روی آدرس‌های **https://** یا **localhost** فعال می‌کنند، نه روی فایل باز شده مستقیم (`file://`). یعنی برای اینکه دکمه‌ی «نصب» ظاهر شود و حالت آفلاین کار کند، باید این پوشه را روی یک هاست آپلود کنید.

## روش‌های ساده‌ی میزبانی (رایگان)
کل پوشه `pwa` را همان‌طور که هست (با ساختار فایل‌ها) آپلود کنید:
1. **Netlify Drop** → app.netlify.com/drop — کشیدن و رها کردن پوشه
2. **GitHub Pages** → محتوای پوشه را در یک ریپازیتوری بریزید و Pages را فعال کنید
3. **Vercel / Cloudflare Pages** → آپلود مستقیم پوشه

## نصب روی گوشی
- **اندروید (Chrome):** بعد از باز کردن آدرس، پیام «افزودن به صفحه اصلی» خودکار ظاهر می‌شود، یا از منوی ⋮ گزینه‌ی Install app را بزنید.
- **آیفون (Safari):** از منوی Share، گزینه‌ی «Add to Home Screen» را بزنید. (در مرورگرهای دیگر آیفون مثل Chrome این گزینه فعال نیست — Safari الزامی است.)

## تغییر آیکون در آینده
اگر بعداً خواستید آیکون برنامه را عوض کنید، فقط فایل‌های داخل `icons/` را با همان نام و همان سایز جایگزین کنید.
